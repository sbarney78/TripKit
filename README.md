TripKit

TripKit is a comprehensive trip‑planning and packing assistant for Android.
Originally designed for camping but flexible enough for any adventure, it helps travelers manage gear, plan meals, and organize itineraries — all in one place.

🚀 Features

📦 Smart Inventory & Packing Lists

• 	Master Inventory
Maintain a permanent “Master List” of gear that can be reused for every trip.
• 	Container Support
Organize items into containers (e.g., Fridge, Tool Box, Dry Food Bin).
• 	Two‑Way Packing Sync
• 	Checking all items in a container automatically marks the container as Done.
• 	Checking a container automatically marks all items inside as packed.
• 	Progress Tracking
Visual indicators show how much of your list is packed.

🍴 Meal Planning & Ingredients\

• 	Menu Planner
Plan meals day‑by‑day (Breakfast, Lunch, Dinner, Snacks).
• 	Ingredient Management
Categorize ingredients into groups (Dairy, Meat, Pantry) for efficient shopping.
• 	Alphabetical Sorting
Ingredient and group lists stay automatically sorted.
• 	PDF Export
Generate a clean, printable PDF of your trip menu.

🗺️ Trip Itinerary

• 	Activity Tracking
Schedule activities by day and time.
• 	Detail‑Rich Entries
Log locations, notes, and estimated costs for each stop.
• 	Dynamic Sorting
Itinerary items automatically order themselves by date and time.

📄 PDF Document Generation

• 	Inventory PDF
Generate a packing list PDF to check off items physically.
• 	Menu PDF
Cleanly formatted meal plans ready for the camp kitchen.
• 	Built‑In Viewer
View and print documents directly inside the app.

🛠️ Technical Stack

• 	Language: Kotlin
• 	UI Framework: Jetpack Compose (Material 3)
• 	Local Database: Room (offline‑first)
• 	Architecture: MVVM
• 	Async Handling: Kotlin Coroutines & Flow
• 	PDF Rendering: Android PDFDocument API
